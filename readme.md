# HeliX Lib

Private Roblox UI library rebuilt around a black-obsidian, electric-cyan control system.

## Visual direction

- Obsidian surfaces, raised dark cards, narrow borders, bright ice-white text
- Electric cyan by default, with accent variants for solar, verdant, whiteout, and redline looks
- Compact rounded modules, thin tactical separators, polished HUD motion
- Gotham typography, premium contrast, and restrained glow instead of oversized toy-like controls

## Included themes

- `HeliX`
- `SolarFlare`
- `Verdant`
- `Whiteout`
- `Redline`

Legacy HeliXLib theme names still resolve for compatibility, but the default identity is now HeliX.

## Private overrides

- `HELIX_ASSET_ID` to replace the main ScreenGui asset
- `HELIX_KEY_ASSET_ID` to replace the key-system ScreenGui asset
- `HELIX_ASSET_BASE_URL` to replace the fallback image pack source

## Example

See `example.lua` and replace the placeholder loader URL with your private distribution endpoint.
